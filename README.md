# Django Todo App

A task management web application built using Django.

## Description
This project is a Django-based Todo application that supports user authentication
and CRUD operations. Users can sign up, log in, and manage their own tasks securely.
Each task is linked to the authenticated user.

## Features
- User signup, login, and logout
- Secure authentication using Django built-in auth
- Create, update, delete tasks
- Tasks linked to logged-in users
- Access control using login_required

## Tech Stack
- Backend: Django
- Database: SQLite
- Frontend: HTML, CSS
- Version Control: Git & GitHub
